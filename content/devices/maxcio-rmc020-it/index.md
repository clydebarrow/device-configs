## ESP-02S GPIO Pinout

| Pin    | Function       |
| ------ | -------------- |
| RX     | Button         |
| GPIO4  | HLW8012 CF Pin |
| GPIO5  | HLWBL CF1 Pin  |
| GPIO12 | HLWBL SEL Pin  |
| GPIO13 | Led            |
| GPIO14 | Relay          |
