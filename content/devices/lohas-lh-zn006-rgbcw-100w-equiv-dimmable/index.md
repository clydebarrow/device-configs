## GPIO Pinout

| Pin       | Function      |
| --------- | ------------- |
| GPIO13    | Data channel  |
| GPIO15    | Clock channel |
| Channel 0 | CW0 channel   |
| Channel 1 | CW1 channel   |
| Channel 2 | CW2 channel   |
| Channel 3 | Blue channel  |
| Channel 4 | Green channel |
| Channel 5 | Red channel   |
