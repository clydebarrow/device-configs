## GPIO Pinout

| Pin    | Function                             |
| ------ | ------------------------------------ |
| GPIO0  | Blue LED 1                           |
| GPIO2  | Blue LED 2                           |
| GPIO4  | Push Button 2 (HIGH = off, LOW = on) |
| GPIO12 | Relay 1                              |
| GPIO13 | Push Button 1 (HIGH = off, LOW = on) |
| GPIO14 | Status LED                           |
| GPIO15 | Relay 2                              |
