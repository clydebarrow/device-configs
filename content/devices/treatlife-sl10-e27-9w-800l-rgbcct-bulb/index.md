## Overview

Treatlife SL10 PCB differs from the previous ESP based SL10 that now has BK7231T chip (W8P) and using sm2135 led driver.

## GPIO Pinout

| Pin    | Function                           |
| ------ | ---------------------------------- |
| GPIO8  | SM2135 Data Pin  |
| GPIO7  | SM2135 Clock Pin          |
