## Overview

Generation 3 of Shelly Mini. With Relay, but no Powermeter.

## GPIO Pinout

| Pin   | Function |
| ----- | -------- |
| GPI00 | LED      |
| GPI01 | Button   |
| GPI03 | NTC      |
| GPI07 | Relay    |
| GPI10 | Switch   |

## Serial Pinout

The UART Pinout is the same as other Shelly Mini.
| Pin     | Colour |
| ------- | ------ |
| Reset   | Brown  |
| 3v3     | Red    |
| RX      | Blue   |
| TX      | Yellow |
| BootSEL | Purple |
| GND     | Black  |
