## GPIO Pinout

| Pin    | Function                           |
| ------ | ---------------------------------- |
| GPIO14 | Push Button (HIGH = off, LOW = on) |
| GPIO05 | Relay                              |
| GPIO12 | Red LED (Used for status)          |
| GPIO04 | Blue LED (Used for active state)   |
