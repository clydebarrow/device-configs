## Overview

This is effectively a Shelly Plus 1PM Mini, but without the relay and the switch.

## GPIO Pinout

| Pin   | Function     |
| ----- | ------------ |
| GPI03 | NTC          |
| GPI06 | BL0942 TX    |
| GPI07 | BL0942 RX    |

## Serial Pinout

The UART Pinout is the same as the Shelly Plus 1PM Mini.
