## Overview

RGBW smart light bulb, A19 shape, E27 base, RGB colors + cold white, 100-240V AC 50/60Hz, natively Tuya/Smart Life, works with Tuya-convert to flash to ESPHome.

## GPIO Pinout

| Pin    | Function         |
| ------ | ---------------- |
| GPIO13 | my9231 data pin  |
| GPIO15 | my9231 clock pin |
