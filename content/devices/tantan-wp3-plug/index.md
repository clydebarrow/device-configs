## Overview

The GPIO pinout was learned from [Blakadder Tasmota](https://templates.blakadder.com/tantan_WP3.html) documentation and adapted through trial and error.

## GPIO Pinout

| Pin    | Function            |
| ------ | ------------------- |
| GPIO04 | Button              |
| GPIO12 | Red LED (inverted)  |
| GPIO13 | Blue LED (inverted) |
| GPIO14 | Relay               |
