## GPIO Pinout

| Pin    | Function              |
| ------ | --------------------- |
| GPIO14 | Onboard LED           |
| GPIO22 | Onboard Button        |
| GPIO25 | PWM 1 (R)             |
| GPIO26 | PWM 3 (B)             |
| GPIO27 | PWM 2 (G)             |
| GPIO4  | PWM 4 (W)             |
| GPIO36 | Input 1               |
| GPIO37 | Input 2               |
| GPIO38 | Input 3               |
| GPIO39 | Input 4               |
| GPIO33 | NTC Temp Sensor       |
| GPIO34 | Voltage Monitor       |
| GPIO35 | OpAmp Current Monitor |

## Plus addon interface

Use it for flashing. Connect EN pin to the RESET pin of your USB-UART converter.
