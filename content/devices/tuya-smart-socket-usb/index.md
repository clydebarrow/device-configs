## General Notes

A smart WIFI GPO relay with UBS B & C ports.
The USB ports are not switched and are always powered.
These devices are sold under many brands on Aliexpress.
Contains CB2S BK7231n module.

## GPIO Pinout

| Pin    | Function                   |
| ------ | -------------------------- |
| P6     | Red Status LED (H)         |
| P8     | Button INPUT_PULLUP        |
| P24    | Relay (H)                  |
